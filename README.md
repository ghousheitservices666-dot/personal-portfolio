\# Personal Portfolio Website 🚀



A full-stack personal portfolio application built using React.js, Node.js, Express.js, and MySQL.



\## Features



\- Responsive portfolio design

\- Dynamic project management

\- Contact form with MySQL storage

\- REST API integration



\## Tech Stack



\### Frontend

\- React.js

\- JavaScript

\- CSS



\### Backend

\- Node.js

\- Express.js



\### Database

\- MySQL



\## Project Structure

client - React frontend

server - Node.js backend
